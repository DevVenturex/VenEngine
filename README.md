# VenEngine
VenEngine is a small Game Engine written in Rust using wgpu for Rendering APIs.

---

## Technoligies
- **Winit** for window creation
- **WGPU** for Rendering APIs
- **Pollster** for async functionsality with *wgpu* (Later replaced by Tokio)

## Why another Engine?
I always wanted to write my own engine so I learned OpenGL mainly with Java but Java is in my Opinion not the best language to code games or game engines.

Also I don't like Unity because of the Studio behind it, Unreal because its to big and Godot because I dont like Python or Languages similar to it.
